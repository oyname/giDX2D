Compilereinstellungen für Visual C++ 5.x / 6.x / 7.x

Damit der Visual C++ Compiler die giDX Bibliothek verwenden kann muß der Ort, in dem die giDX² Bibliothek gespeichert wurde, dem Compiler bekannt gemacht werden. Gehen Sie auf Extras->Optionen und wählen den Reiter Verzeichnisse. Tragen Sie bei den Include-Dateien den Ort für den Ordner ...gidx/include ein und bei den Bibliothekdateien den Ort für den Ordner ...gidx/lib. Das DirectX SDK 7 muss installiert und auch für das DX-SDK die Compilereinstellungen vorgenommen sein. Obwohl giDX Programme mit int main( ) beginnen ist auch dies eine WIN32 Anwendung. Um eine neue Anwendung zu beginnen gehen Sie auf Datei->Neu. Wählen Sie Win32-Anwendung. Bestätigen Sie mit OK. Fügen Sie Ihrem Projekt das giDX² SDK hinzu. In dem Sie bei Projek->Dem Projekt hinzufügen->Datei alle cpp-Dateien im Ordner giDXSDK in Ihre Projekt einfügen. Jetzt können Sie ihren Code eingeben.
