HFONT createfont(int x, int y, int fnWeight, char* face)

Beschreibung:

Erstellt ein Windowsfont.

Parameter:

Die Funktion gibt das erstellte Font zurück.

int x.............Fontbreite. Bei 0 wird der Standardwert verwendet
int y.............Fonthöhe. Bei 0 wird der Standardwert verwendet
int fnWeight......setzt die Fontstärke.
char* face........Fontname

Konstanten für die Fontstärke:

FW_DONTCARE
FW_THIN
FW_EXTRALIGHT
FW_ULTRALIGHT
FW_LIGHT
FW_NORMAL
FW_REGULAR
FW_MEDIUM
FW_SEMIBOLD
FW_DEMIBOLD
FW_BOLD
FW_EXTRABOLD
FW_ULTRABOLD
FW_HEAVY
FW_BLACK
