Was ist giDX²

Um auf die Hardware zugreifen zu können benötigt man geeignete Schnitstellen wie DirectX oder OpenGL. Die Verwendung dieser APIs erweist sich oft als sehr kompliziert. Damit sich der Spieleentwickler auf das Gameplay konzentrieren kann verwendet man Bibliotheken die diese Arbeit übernehmen.

giDX² ist eine DirectX7 basierende 2D Engine für den Visual C++ Compiler von Microsoft.

Die giDX² Engine soll die Verwendung von DirectDraw7,DirectSound und DirectInput so einfach wie möglich gestalten. Das beginnt mit der Initialisierung von DirectX, setzt sich fort über das Laden und die Ausgabe von Medien bis zur Beendung der Anwendung. giDX² kann Anwendungen im Fenster und im Vollbild darstellen. Auch das wechseln zwischen einer Fenster- und Vollbilddarstellung beherscht giDX². Alle Medien, die in den Speicher geladen werden, werden von giDX² verwaltet und auch um die Speicherfreigabe kümmert sich die 2D Engine. Der Funktionensumfang ist schnell zu erlernen, einfache C Kenntnisse reichen aus um mit der Engine arbeiten zu können.

Haupteigenschaften der giDX² Engine
  • Einfache DirectX Initialisierung
  • Fenster - oder Vollbildanwendungen möglich.
  • Einfaches Laden und Anzeigen von Bitmaps
  • Pixelgenaue Kollisionsabfrage
  • Keine Windows oder DiretcX Kenntnisse erforderlich
  • Speicherverwaltung für Grafik und Sound.
