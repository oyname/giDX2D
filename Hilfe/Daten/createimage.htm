int createimage(LPLPIMAGE lplpimg, int x, int y, int mem)

Beschreibung:

Reserviert Speicher für ein Bild in der Größe x,y im Video- oder Arbeitsspeicher.


Parameter:

Der Rückgabewert sollte ungleich Null sein.

LPLPIMAGE lplpimg....Zeigt auf den erstellten Speicher
int x................Größe in X
int y................Größe in Y
int mem..............gibt an wo das Bild gespeichert werden soll

Mit GXVIDEOMEMORY wird das Bild im Speicher der Grafikkarte erstellt. Mit GXSYSTEMMEMORY wird das Bild im Arbeitsspeicher ertellt.
