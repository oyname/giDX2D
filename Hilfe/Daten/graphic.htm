int graphic(int x,int y, int bpp, int mode = 1)

Beschreibung:

Initialisiert DirectX7 mit der gewünschten Auflösung als Fenster oder Vollbildanwendung. Erst nach einer erfolgreichen Initialisierung sind die Grafik- , Eingabe und Soundfunktionen einsetzbar.

Gängige Auflösungen sind 800x600, 1024x768 oder 1280x1024. Als Farbtiefe ist 16 Bit oder 32Bit zu wählen.


Parameter:

Der Rückgabewert sollte ungleich Null sein. Sonst ist ein Fehler bei der Initialisierung aufgetretten.

int x.......Horizontale Auflösung
int y.......Vertikale Auflösung
int bpp.....Farbtiefe
int mode....GXFULLSCREEN oder GXDEBUG

Der Fenstermodus war für die Fehlersuche gedacht, deshalb lautet die Konstante für eine Fensteranwendung GXDEBUG.
